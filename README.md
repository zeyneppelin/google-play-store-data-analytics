#  Google Play Store Data Analysis  
*Python • Pandas • NumPy • Seaborn • Matplotlib • EDA • Data Cleaning • Feature Engineering*

Bu proje, Google Play Store uygulamalarına ait veri seti üzerinde kapsamlı veri analizi gerçekleştirmek amacıyla hazırlanmıştır. Çalışma kapsamında veri temizleme, ön işleme, keşifsel veri analizi ve veri görselleştirme teknikleri uygulanmıştır.


# Projenin Amacı
- Uygulama kategorilerini analiz etmek  
- Puan ve indirme sayısı ilişkisini incelemek  
- Eksik ve hatalı verileri temizlemek  
- Kullanıcı davranışlarını veri üzerinden yorumlamak  


# Veri Temizleme
- Eksik değer yönetimi  
- Veri tipi dönüşümleri  
- Kategorik verilerin düzenlenmesi  
- Sayısal alan hatalarının düzeltilmesi  


# EDA (Keşifsel Veri Analizi)
- Kategori dağılımlarının incelenmesi  
- Puan histogramları  
- İndirme sayısı–puan ilişkisi  
- Korelasyon analizleri  


# Veri Görselleştirme
Matplotlib ve Seaborn kullanıldı:
- Bar chart  
- Histogram  
- Scatter plot  
- Box plot  


#Feature Engineering
- Kategorik verilerin encode edilmesi  
- Karmaşık verilerin temizlenmesi  
- Modellemeye uygun hale getirme  


# Kullanılan Teknolojiler
- Python  
- Pandas  
- NumPy  
- Seaborn  
- Matplotlib  
- Jupyter Notebook  


# Sonuç
Bu proje veri bilimi alanında temel beceriler kazanmak, veri işleme–analiz süreçlerini öğrenmek ve Python kütüphanelerini aktif şekilde kullanmak amacıyla geliştirilmiştir.
